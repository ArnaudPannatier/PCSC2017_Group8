Detailed Description

Preconditioned Conjugate Gradient Method.

This class implements the Preconditioned Conjugate Gradient Method for solving a system of linear equations, namely those whose matrix is symmetric and positive-definite. https://en.wikipedia.org/wiki/Conjugate_gradient_method#The_preconditioned_conjugate_gradient_method

#include "PCConjugateGradientDescent.h"
PCConjugateGradientDescent PCconjSolver(A, B, x_0, M);
Vector X = PCconjSolver.solve();
