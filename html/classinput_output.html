Matrix inputOutput::readFromText ( string  filename)
static

Reads a matrix from a text file.

Parameters
filenameeither relative or absolute path to the text file
Returns
a matrix

This method reads a matrix from a text file. Note that column entries must be separated by commas and row entries separated by a newline. The method will give a warning if the number of entries in each row are unequal. It will fill missing entries with the value 0.

#include "inputOutput.h"
inputOutput io;
Matrix A = io.readFromText("A.txt");
Matrix b = io.readFromText("b.txt");
