7 # MATH-458 - PCSC - Linear solvers
8 
9 This C++ project consist in solving linear system _Ax = b_ by different methods.
10 
11 7 solvers are implemented : LU decomposition, Cholesky decomposition, Conjugate Gradient Descent, Preconditioned Conjugate Gradient Descent, Jacobi, Gauss-Seidel, and Richardson method.
12 The first two solvers are direct methods and the others are iterative methods.
13 
14 This project is a part of the course : "*Programming Concepts in Scientific Computing*" given at EPFL by **Guillaume Anciaux** in fall 2017.

116 Vector b1 = Vector(io.readFromText("B1.txt"));

125 // Example of running the conjugate gradient method

127 ConjugateGradientDescent conjSolver(A2, B2, x_0);
128 Vector X_sol_conj = conjSolver.solve();

142 ## List of Features
143 ### Available solvers
144 
145 Below are the list of implemented solvers and their constructors.
146 
147 #### Direct solvers :
148 - LU
149 
150  Implements the LU decomposition of a matrix based on Crout's algorithm that returns a lower triangular matrix and a unit upper triangular matrix.
151 
152  https://en.wikipedia.org/wiki/Crout_matrix_decomposition
153 
154 ```c++
155 LU(A, B, opts[X_0])
156 ```
157 
158 
159 - Cholesky
160 
161 Implements the Cholesky decomposition of a matrix that returns a lower triangular matrix and its transpose. It's a special case of LU decomposition when the matrix A is symmetric.
162 
163 https://en.wikipedia.org/wiki/Cholesky_decomposition
164 
165 ```c++
166 Cholesky(A,B, opts[X_0])
167 ```
168 
169 #### Iterative solvers :
170 - Conjugate Gradient Descent
171 
172 Implements the method for conjugate gradient descent. Which has the property to converge when the matrix is symmetric definite positive. In the other cases, the convergence can appear as well but with no guarantee.
173 
174 https://en.wikipedia.org/wiki/Conjugate_gradient_method
175 
176 ```c++
177 ConjugateGradientDescent(A, B, opts[ X_0, epsilson, max_iter ])
178 ```
179 
180 - Preconditionate Gradient Descent
181 
182 Implements the method for preconditioned gradient descent. Preconditioning will ensure that the matrix has a smaller condition number. This should lead to a faster and numerically more stable convergence.
183 
184 https://en.wikipedia.org/wiki/Conjugate_gradient_method
185 
186 ```c++
187 PCConjugateGradientDescent(A,B, P, opts[ X_0, epsilson, max_iter ])
188 ```
189 
190 
191 - Gauss Seidel
192 
193 Implements the Gauss-Seidel method for solving a system of linear equations.
194 
195 https://en.wikipedia.org/wiki/Gauss%E2%80%93Seidel_method
196 
197 ```c++
198 GaussSeidel(A, B, opts[ X_0, epsilson, max_iter ])
199 ```
200 
201 
202 - Jacobi
203 
204 Implements the Jacobi method for solving a system of linear equations
205 
206 https://en.wikipedia.org/wiki/Jacobi_method
207 
208  ```c++
209 Jacobi(A, B, opts[ X_0, epsilson, max_iter ])
210  ```
211 - Richardson
212 
213 Implements the Richardson Method for solving a system of linear equations
214 
215 https://en.wikipedia.org/wiki/Modified_Richardson_iteration
216 
217 
218 ```c++
219 Richardson(A, B, omega, opts[ X_0, epsilson, max_iter ]);
220 ```

222 ### Types
223 - Matrix
224 
225 Class that represents Matrices. They can be of any size. An instance of this class can be created from `vector<vector<double>>`, ` initializer_list ` and can also be read from files using the reader.
226 
227 Operator +,-, *double, *Matrix, =,\[ \],() have been overloaded to allow pratical computation.
228 Operator << and >> have been overloaded to allow practical display and practical translation from string.
229 
230 Some typical methods for matrices as `Transpose()`, ` Inverse() ` ... also have been implemented.
231 
232 - Vector
233 
234 Inherit from Matrix class. Represent vector type.
235 
236 Some practical methods for vectors as scalar product : ` dot()`, norm induced from scalar product : ` norm() `, ... also have been implemented.
237 
238 ### Reader and writer
239 
240 A class reader is provided in order to create an instance of Matrix from a .txt file.
241 
242 ```c++
243 Matrix A = inputOutput::readFromText("A.txt");
244 ```
245 
246 A writer was also implemented if one want to output the results in a file instead of printing it in the terminal.
247 
248 ### Solver Factory
249 
250 A solver factory is implemented to create the Solver that the user want. It takes as input the parameters that the user provides either in the _arguments_ or in the _cin_ and return a pointer to a Solver.
251 
252 The resolution is then made taking use of the polymorphism of the class.
253 
254 A templated function `stoClass` which translate string to object have also been implemented. It was practical for the solver factory.
